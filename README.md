# LoFi Text Editor

Implementazione di un text editor Local-First.

## Stack

**Frontend**: React 18 (Vite)  
**Backend**: Express + Prisma + Keycloak  
**Database**: PostgreSQL

**Local-First**: [Automerge](https://github.com/automerge) + Custom Network/Storage
